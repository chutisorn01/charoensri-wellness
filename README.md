# Charoensri Wellness - เจริญศรีนวดแผนไทย

ระบบจัดการข้อมูลสปาและเจริญศรีนวดแผนไทย ครบวงจร ทั้งระบบหน้าบ้าน (Customer-facing Homepage) และระบบหลังบ้าน (Admin Panel Dashboard)

## โครงสร้างโครงการ (Project Structure)
* `backend/` - Node.js / Express.js / MongoDB Mongoose API
* `frontend/` - React (TypeScript) / Vite / CSS / Framer Motion

## การติดตั้งและการเริ่มพัฒนา (Setup & Running)

### 1. ระบบหลังบ้าน (Backend API)
```bash
cd backend
npm install
npm run dev
```

### 2. ระบบหน้าบ้าน (Frontend Client)
```bash
cd frontend
npm install
npm run dev
```
